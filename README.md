# 🛒 Flipkart Manual Testing Project

## 📌 Project Overview

This project demonstrates **manual testing performed on the Flipkart application**, covering key e-commerce functionalities. The testing activities include writing and executing test cases, identifying defects, and validating expected results.

## 🎯 Objective

* To validate core functionalities of the Flipkart website
* To ensure the system behaves as expected under different scenarios

## 🧪 Test Case Details

The test cases in this project include:

* Test Case ID
* Test Scenario
* Test Description
* Preconditions
* Test Steps
* Test Data
* Expected Result
* Actual Result
* Status (Pass/Fail)
